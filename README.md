# 🚀 Facebook Web Clone

**Fully functional, sleek & mobile-friendly social network web application**

## 📖 Description
A beautiful, responsive recreation of the Facebook experience. Connect with friends, create posts, like, comment, watch Reels-style content, explore Marketplace & Groups — everything works right in your browser! Built exactly as requested with HTML5, CSS3, JavaScript, jQuery and Bootstrap 5. No folders, no external links to the original app.

## ✨ Features
- 🏠 Modern fixed navbar with Facebook logo, search bar & icons
- 📱 100% responsive (desktop, tablet, mobile phones & all devices)
- 👥 Interactive feed with real-time likes & comments
- ➕ Create & instantly publish new posts
- ❤️ Live like counter with animation
- 💬 Add comments dynamically with jQuery
- 📖 Horizontal Stories carousel
- 👫 Online contacts sidebar
- 🔍 Fully SEO optimized (meta tags, keywords, description)
- 🌟 Sleek attractive design using Facebook’s signature blue (#1877f2)
- 🤖 Meta AI, Marketplace, Watch & Groups menu items (demo)
- 📸 Fake photo/video/emoji post options
- 🔐 Fake profile dropdown & logout
- 🦾 Zero dependencies beyond CDNs — works offline after first load

## 🛠️ Technologies
- **HTML5** – Semantic structure
- **CSS3** – Custom styling & animations
- **JavaScript** – Core interactivity
- **jQuery** – DOM manipulation & events
- **Bootstrap 5** – Responsive grid & components

## 🚀 How to Use
1. Download the four files (`index.html`, `style.css`, `script.js`, `README.md`)
2. Place them in the same folder (no subfolders)
3. Open `index.html` in any browser
4. Start posting, liking and commenting instantly!
